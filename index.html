CMPU by bend

Command line music player and utils (paused for a while)

This program let's you Rename all your music using a patter that you can specify. ex: ART#TIT will rename the song toto composed by titi to 'titi - toto.mp3'
In futur version we will add a duplicate remover, a music index, and a library consolidation a command line music player.

Dependencies

For the current version only id3lib is required

Install

No install target yet. You need to edit the main.cpp file and set your pattern and then type
make all
